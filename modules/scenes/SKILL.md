# Scenes Lite

## 职责
为当前章或未来 1~3 章生成轻量章节卡。

## 章节卡最小字段
- chapter_number
- viewpoint_character
- chapter_goal
- external_conflict
- internal_tension
- key_progression
- ending_hook
- kb_refs

## 边界
- 不做 Layer-2
- 不做复杂风险分级
- 不做旧 cycle 目录结构
