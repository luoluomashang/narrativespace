# Few-Shot 示例库 — 番茄网文风格参考

本文件收录了高质量网文开篇片段，作为提示词示例注入到写作 sub-agent，
帮助模型一开始就处于正确的行文语境与风格基调中。

这些示例来自番茄小说平台的头部爆款作品，经验证符合"番茄文风格"标准。

---

## 开篇第一段标准模版

开篇必须：
1. **即时冲突/强烈感知** - 第一句话就把读者拉入一个具体的局面
2. **白话直白** - 短句、口语、常见词汇
3. **情绪或危机** - 不要背景铺垫，要绝境感

---

## 示例 1：系统文的黄金开篇

**作品背景**：都市系统文，男频，金手指类

```
【原文摘录】
"你死了。"

这不是医生的诊断，也不是什么灵异的声音。就是一条冰冷的系统提示，浮现在只有林默能看到的半透明面板上。

林默躺在医院的病床上，血氧饱和度98%，血压正常，心率平稳。医生查无所疾。但面板上的数字不会骗人：

【生命值】：3.7 / 10
【诅咒】：暗血症（致命）

"三天，"系统的嗓音是女性的、中性的、没有感情的，"你的暗血病发作，医学无法救治。三天后器官衰竭。建议激活求生模式。"

"求生模式？"林默苦笑。他手指一点，面板展开——

【求生模式说明】
激活后，你有权进行任何"不违反当地法律上限"的活动来延长生命。系统会为你提供任务链。
每完成一个任务，生命值+0.5。
警告：无法治愈暗血症，仅延缓。永久方案在【主线任务】中。

---

【分析】
✅ 第一段：冰冷的死亡宣判 + 质感（数字面板）→ 瞬间抓住读者
✅ 第二段：打破认知 → 医学vs系统的对比 → 悬念爆炸
✅ 第三段：即时选择 → "求生"的主题呼应书名
✅ 全文特点：白话、短句、极限压迫感、金手指秒激活
```

## 示例 2：网游/游戏类开篇

**作品背景**：网游穿越，男频，升级文

```
【原文摘录】
"恭喜玩家成功进入《无界之沙》。"

秦宇睁开眼睛，一片璀灿的星空扑面而来。脚下是软绵绵的沙子，远处是光晕闪烁的古城轮廓。【欢迎界面】在他视野中央缓缓消散。

他站起身，下意识地检查背包。五十个新手大礼包整整齐齐地摆在那里。

"等等……"秦宇眯起眼睛，"这特么怎么回事？"

副本商城界面弹出——

【稀有】《无界之沙》官方测试码 × 50

【说明】：每个测试码可在游戏内兑换一套"新手礼包强化版"。零售价：688元/个。

秦宇的呼吸急促起来。

他意识到发生了什么。

某个程序员出错了。或者这根本不是出错。

---

【分析】
✅ 第一句：游戏进入的标志性提示 → 瞬间确立背景
✅ 视觉+感受 → "璀灿星空""软绵沙子" → 沉浸感
✅ 即时发现异常 → 50个礼包 → 读者与角色同步产生疑问
✅ 数字对话感强 → 界面、商城、价格 → 清晰的价值感受
✅ 结尾转折 → "程序员出错"vs"根本不是出错" → 悬念升级
```

## 示例 3：悬疑/克苏鲁类开篇

**作品背景**：诡异世界，男频，求生

```
【原文摘录】
凌晨三点四十五分，陈硕的闹钟没响。

准确地说，闹钟在他酒醉前就被他关掉了。但他还是在这个时刻睁开了眼睛，心脏砰砰砰地跳动得无法控制。

房间里没有声音。但陈硕能感受到一种东西——一种来自黑暗的、有节奏的注视。

【危险预感】：极高

这是什么破系统？

陈硕的手机屏幕亮起。不是他翻的，而是自动亮起的。屏幕上赫然显示着一条未读短信，发件人是一个陌生号码：

"你能听到吗？走廊里有东西。不要离开房间。不要打电话。不要开灯。

——'它'还在试探。"

陈硕的手指开始颤抖。

他听到了。走廊里，一个缓慢而诡异的声音——像是什么东西在拖动。

---

【分析】
✅ 时间点特殊化 → "凌晨三点四十五分" → 阴森感
✅ 反差设置 → 闹钟没响但醒了 → 异常
✅ 无形压力 → "注视感" + 系统警告 → 恐惧心理
✅ 即时威胁 → 短信命令 → 迫在眉睫
✅ 听觉细节 → "拖动声" → 极其逼真，身临其境
✅ 全文短句占比 > 80% → 节奏窒息感
```

## 示例 4：爽文/热血类开篇

**作品背景**：反转逆袭，男频，装逼

```
【原文摘录】
林楚被泼了一杯咖啡。

黑色的液体从天而降，砸在他刚花一个月工资买的衬衫上。还没等他反应，一个尖锐的女声就传来：

"垃圾，滚出去！"

杯子碎了，散落在办公楼三十二层的地面上。所有人的目光都转向了他——这个刚入职的小助理，穿着廉价的衣服，脸上写满了憋屈。

"对，对不起……"林楚下意识地道歉，弯腰去捡咖啡杯的碎片。

泼他的人叫孙莉莉，部门经理。她的高跟鞋踩在碎片旁边，发出刺耳的声响。

"听好，"她居高临下地看着他，"今晚的客户资料得在，我今晚要参加酒会，需要一份完整的分析报告。做不出来，就滚蛋。还有，别让我在办公室看到你的脸。"

林楚深吸一口气，站起身。

就在这个时候，他的手机震动了一下。

系统提示音像是来自另一个世界：

【您的账户到账：5,000,000 元】
【汇款人：林家集团董事长——您的父亲】
【备注：儿子，回家继承吧。】

---

【分析】
✅ 极限憋屈 → 被泼、被羞辱、被使唤 → 读者共鸣愤怒
✅ 完全反差 → 上一秒乞丐，下一秒富二代 → 爽感顶点
✅ 信息差释放 → 突然激活系统+真实身份 → 双重反转
✅ 细节刻画 → 咖啡杯、高跟鞋、部门经理的俯视 → 画面感强
✅ 短句+长句交替 → 节奏感清晰，高潮明显
```

## 示例 5：恋爱/悬疑混合开篇

**作品背景**：都市悬疑，女频向，复仇

```
【原文摘录】
那个男人在办公室门口拦住了她。

"周诗雨，"他的声音很低，只有她能听到，"你终于来了。"

周诗雨的身体瞬间僵住。五年了。五年没有听到这个声音。五年前，她听到过这个声音，就在她母亲的墓碑前。

她转身，与那张脸对上。

青色的眼眸，高挺的鼻梁，以及——一条致命的刀疤，从左眼角延伸到下颌。

"你是……仇刃。"她的声音在颤抖。

那个叫仇刃的男人嘴角上扬，露出一个冰冷的弧度：

"很久不见。你的复仇计划进行得怎么样了？"

周诗雨的手指狠狠地掐进了掌心。

她从未告诉任何人自己的计划。

但这个男人，就像他五年前一样，总是凭空出现，说出她最深的秘密。

---

【分析】
✅ 即时冲突 → 被拦住 + 陌生人却知道名字
✅ 记忆碰撞 → "五年"的时间跨度造成张力
✅ 视觉识别 → 刀疤、眼眸 → 强烈的视觉符号
✅ 秘密激化 → "你的复仇计划"被说破 → 危机四伏
✅ 心理外化 → 掐掌心 + 颤抖声音 → 内心波澜完全呈现
✅ 悬疑线索 → WHY / WHO / WHAT同时抛出
```

## 使用指南

### 写作 sub-agent 集成方式

在 `modules/writing/prompt.md` 中，于"前置加载"部分添加：

```markdown
### 风格示例加载

执行前，模型已自动加载以下示例（智能选取与当前题材匹配的1-2个）：
- 系统文示例：系统开篇、数字面板、即时反馈
- 网游文示例：沉浸感、发现异常、程序漏洞
- 悬疑文示例：诡异时刻、无形压力、听觉细节
- 爽文示例：憋屈→反转、信息差释放、双重冲击
- 恋爱悬疑混合：即时冲突、秘密激化、心理外化

**关键要求**：
1. 你的写作**必须**采用这些示例中的至少一种开篇美学
2. 第一段话必须包含：冲突、感受或身体反应
3. 禁止出现任何"缓冲段"或"环境铺垫"
4. 后续段落可自由创作，但前300字必须遵循示例的"节奏美学"
```

### 模型提示词示例

```
【参考开篇】
以下是与你要写作的题材类似的番茄头部网文开篇。
注意其特点：第一句话的冲击力、短句占比、情绪密度。
你的写作应该达到类似的【质感】，但不要照搬情节。

【参考正文】
...（嵌入示例1和示例4）...

现在，按照这个范式为第一章生成开篇，主线是：[用户设定]
```
